About this project

Hope you are doing well in these odd times! I get inspiration from the Hopkins COVID19 update system (https://coronavirus.jhu.edu), so I wrote this to indicate the number of cases reported globally from 01/22/2020 to 04/22/2020 by region. Hope we can get back to normal soon!

How it is made

  • Acquired global date from JHU CSSE at https://github.com/CSSEGISandData/COVID-19
  • Tidy up the raw data (so that the format matches my requirement)
  • Wrote the code so that the case report of a region can show as a bar plot.
